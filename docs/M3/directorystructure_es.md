# Estructura de directorios de los componentes de Material

Todos los Material Components están ubicados en **[lib/](https://github.com/material-components/material-components-android/tree/master/lib)**.

Las clases en la librería están separadas en directorios que son específicos para cada componente, por ejemplo:

*   [appbar/](https://github.com/material-components/material-components-android/tree/master/lib/java/com/google/android/material/appbar/)
*   [bottomnavigation/](https://github.com/material-components/material-components-android/tree/master/lib/java/com/google/android/material/bottomnavigation/)
*   [button/](https://github.com/material-components/material-components-android/tree/master/lib/java/com/google/android/material/button/)

Las clases en los directorios de los componentes comprenden la API pública; estas pueden ser usadas directamente en tus aplicaciones. Las clases en **internal/** son parte de la API protegida y se usan para dar soporte a las clases de la API pública.
