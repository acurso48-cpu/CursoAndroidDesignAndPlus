# Aplicación de Catálogo

Con demostraciones que cubren una amplia gama de temas, el Catálogo de Material Components demuestra cómo los componentes y principios de Material Design se comportan en dispositivos reales a través de diferentes niveles de API. El Catálogo de Material Components también proporciona ejemplos de código funcionales para los desarrolladores.

Para probar la aplicación Catálogo de MDC, puedes ejecutar el módulo `catalog` en Android Studio o ejecutar el siguiente comando de Gradle:

```sh
./gradlew :catalog:installDebug
```
