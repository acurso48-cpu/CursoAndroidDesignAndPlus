# Guía de Temas

La creación de temas de Material (Material Theming) es la capacidad de personalizar sistemáticamente Material Design para reflejar mejor la marca de tu producto.

Una vez que hayas actualizado el tema de tu aplicación para que herede de un tema de Material 3, como se explica en la guía de [Primeros Pasos](getting-started.md), puedes empezar a aprovechar los sistemas de temas para toda la aplicación que están integrados en Material Components para Android.

Para obtener información sobre los sistemas individuales de Material Theming, echa un vistazo a las siguientes guías:

*   [Temas de Color](theming/Color.md)
*   [Temas de Tipografía](theming/Typography.md)
*   [Temas de Forma](theming/Shape.md)
*   [Temas de Movimiento](theming/Motion.md)

Además, echa un vistazo a nuestra guía de [Tema Oscuro](theming/Dark.md) para obtener información sobre cómo aplicar el [sistema de tema oscuro de Material](https://material.io/design/color/dark-theme.html) a tu aplicación.
