# Botones

Los [Botones](https://m3.material.io/components/all-buttons) permiten a los usuarios realizar acciones y tomar decisiones con un solo toque.

Para obtener la documentación completa sobre el uso, comportamiento y atributos de los diferentes tipos de botones de Material, consulta:

*   [Botones comunes](https://github.com/material-components/material-components-android/tree/master/docs/components/CommonButton.md)
*   [Grupos de botones](https://github.com/material-components/material-components-android/tree/master/docs/components/ButtonGroup.md)
*   [Botón dividido (Split button)](https://github.com/material-components/material-components-android/tree/master/docs/components/SplitButton.md)
*   [Grupos de botones de palanca (Toggle button groups)](https://github.com/material-components/material-components-android/tree/master/docs/components/ToggleButtonGroup.md)
*   [Botón de icono](https://github.com/material-components/material-components-android/tree/master/docs/components/IconButton.md)
*   [Botones de acción flotantes (FABs)](https://github.com/material-components/material-components-android/tree/master/docs/components/FloatingActionButton.md)
*   [Botones de acción flotantes extendidos (FABs extendidos)](https://github.com/material-components/material-components-android/tree/master/docs/components/ExtendedFloatingActionButton.md)
